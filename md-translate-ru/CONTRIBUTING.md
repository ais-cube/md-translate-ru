# Как внести вклад

Спасибо за интерес к проекту!

## Сообщить о проблеме

Откройте [Issue](../../issues) с описанием:
- Что произошло
- Что ожидалось
- Версия Python и ОС
- Лог ошибки (если есть)

## Предложить улучшение

1. Форкните репозиторий
2. Создайте ветку (`git checkout -b feature/my-idea`)
3. Внесите изменения
4. Отправьте Pull Request
