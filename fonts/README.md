# Шрифты / Fonts

Шрифты для генерации PDF с полной поддержкой кириллицы.

## Состав

| Шрифт | Назначение | Лицензия |
|---|---|---|
| DejaVu Sans | Основной текст, заголовки | [Bitstream Vera + Arev](https://dejavu-fonts.github.io/License.html) |
| DejaVu Sans Mono | Блоки кода | то же |
| Liberation Sans | Совместимость с Arial/Helvetica | [SIL OFL 1.1](https://scripts.sil.org/OFL) |

## Почему локальные шрифты

- Гарантированная работа на любой машине без установки системных шрифтов
- Одинаковый результат на macOS, Windows, Linux
- Полная поддержка кириллицы, латиницы, математических символов

## Добавить свой шрифт

Положи `.ttf` файлы в эту папку и укажи путь в `convert.py`:

```python
FONT_DIR = Path(__file__).parent / "fonts"
```

Скрипт автоматически подхватит все `.ttf` из этой папки.
