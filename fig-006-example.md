## Тип изображения

Схема

## Краткое описание

Архитектурная схема паттерна Prompt Chaining. Показывает, как пользователь отправляет серию промптов нескольким агентам, где выход каждого агента становится входом для следующего в цепочке.

## Структура изображения

Схема построена вертикально-горизонтально с тремя уровнями:

- Слева: фигура пользователя (User), от которой отходят две стрелки вправо
- Нижний поток: Prompt 1 -> Agent 1 (стрелка вверх к следующему уровню)
- Средний уровень: многоточие (...) - обозначает промежуточные звенья цепочки
- Верхний поток: Prompt n -> Agent n (стрелка вправо и вверх к Output)
- Вверху: блок Output со стрелкой возврата к пользователю

Каждый блок Prompt содержит иконки: угловые скобки (>) и символ sparkle. Каждый блок Agent содержит иконку головы с чипом AI.

## Текстовые элементы
